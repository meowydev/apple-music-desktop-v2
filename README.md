# Installation

Get the app in [Releases](https://github.com/meowydev/apple-music-desktop-v2/releases).

No additional dependencies are required

# Developing

## Prerequisites
node.js 16 or 18 is required.

## Preparing the development enviroment

### Clone the repo

```bash
git clone https://github.com/meowydev/apple-music-desktop-v2.git
cd apple-music-desktop-v2
```

### Install dependencies

```bash
npm install
```

### Run the app

```bash
npm run start
```
